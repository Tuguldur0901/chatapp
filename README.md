# ⚡ LiveChat — Socket.io Real-time Chat

Бодит цагийн олон хэрэглэгчийн чат систем.

## Технологи
- **Backend**: Node.js + Express + Socket.io
- **Frontend**: Vanilla JS + CSS (хөгжилтэй дизайн)

## Суулгах

```bash
npm install
```

## Ажиллуулах

```bash
npm start
```

Дараа нь браузерт нээнэ:
```
http://localhost:3000
```

## Өрөөнүүд
- `#general` — Ерөнхий яриа
- `#tech` — Технологи
- `#random` — Дурын сэдэв
- `#design` — Дизайн

## Боломжууд
- Нэр оруулж чатад нэгдэх
- 4 өрөөний аль нэгд орох
- Бодит цагийн мессеж солилцох
- Typing indicator (хэн бичиж байгааг харах)
- Онлайн хэрэглэгчдийн жагсаалт
- Өрөө солих
- Чатаас гарах

## Хөгжүүлэх

```bash
npm run dev  
```
